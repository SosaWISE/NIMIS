Javascript

RestCake services can dynamically create javascript clients for you. You just have to include the <script> tag, and then create an instance of the client.
  • JavaScript jQuery client (Don't forget to include js/Kin2Services.js!))


Example: 
<!-- include the js support code first -->
<script type="text/javascript" src="/js/Kin2Services.js"></script>
<!-- and then the generated proxy -->
<script type="text/javascript" src="<#= ServiceBaseUrl #>_js?jquery"></script>

<script type="text/javascript">
	// Create a new instance of the js client
	var svc = new <#= ServiceNamespace #>.<#= ServiceName #>();

	function foo()
	{
		// Call a service method using the js client
		svc.someServiceMethod(arg1, arg2, arg3, function(result)
		{
			// Success! Do something with result...
		}, this); //'this' passed as the callbackScope
	}

	//alternate usage

	function foo()
	{
		// Call a service method using the singleton client instance
		<#= ServiceNamespace #>.<#= ServiceName #>.Instance.someServiceMethod(arg1, arg2, arg3, function(result)
		{
			// Do something with result...
		}, this); //'this' passed as the callbackScope
	}

</script>

Javascript callback signature:
function(result, textStatus, xhr|jqXhr)

If a callbackScope was specified, the callback function will have its 'this' set to callbackScope.
The xhr|jqXhr depends on your version of jQuery. It's xhr pre 1.5, jqXhr for 1.5+. RestCake is just passing along some of the args that jquery returns.
