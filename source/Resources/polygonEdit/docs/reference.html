class Polygon

Extends standart class google.maps.Polygon by methods runEdit() and stopEdit()

Methods

Methods Return Value Description
runEdit(flag:) None Starts editing the polygon. Optional parameter flag indicates the use of ghost markers in the middle of each segment. By default, the flag is true.
stopEdit() None Stops editing Polygon
